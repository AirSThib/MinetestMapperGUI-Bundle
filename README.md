MinetestMapperGUI-Bundle
========================

Contains the Minetestmapper and the MinetestmapperGui as bundle.

Building on Windows
-------------------
Requirements:

- vcpkg https://github.com/Microsoft/vcpkg
- Qt-Framework https://www.qt.io/

Build steps:
just execute the build.ps1 script.
